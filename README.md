# Landing Page - Itaú

Este é um projeto de uma landing page inspirada no Itaú, projetada para treinar habilidades de front-end

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Menu responsivo
- Campo de pesquisa
- Seções informativas sobre os serviços do Itaú
- Links para download do app
